# GotIt - Анимации

Проект использует GSAP для премиум анимаций в стиле "Text Decode".

## Установка зависимостей

```bash
cd frontend
npm install
```

Это установит GSAP автоматически.

## Реализованные анимации

### 1. Text Decode (Декодирование текста)
- **Где используется:** Все заголовки h2 на сайте
- **Эффект:** Буквы появляются в случайном порядке с размытием
- **Компонент:** `DecodeText.js`
- **Фазы:**
  1. Невидимая (opacity: 0, blur: 10px)
  2. Призрачная (opacity: 0.3, слегка размыто)
  3. Активная (opacity: 1, четкая)

### 2. Page Transition (Переходы между страницами)
- **Где используется:** Все страницы (SearchPage, TrackedList)
- **Эффект:** Плавное появление контента
- **Компонент:** `PageTransition.js`

### 3. Stagger Animation (Каскадное появление)
- **Где используется:** Карточки стримеров, грид товаров
- **Эффект:** Элементы появляются по очереди с задержкой
- **Хук:** `useStaggerAnimation.js`

### 4. Background Gradients (Анимированный фон)
- **Где используется:** Глобально на всех страницах
- **Эффект:** Плавающие градиентные шары
- **Компонент:** `BackgroundGradient.js`

## Параметры анимаций

### DecodeText
```jsx
<DecodeText 
  text="Ваш текст" 
  delay={0}         // Задержка перед началом (секунды)
  className=""      // Дополнительные CSS классы
/>
```

### useStaggerAnimation
```jsx
const containerRef = useStaggerAnimation(
  '.element-class',  // Селектор элементов для анимации
  0.2                // Задержка перед началом (секунды)
);
```

## Производительность

- Используется `will-change` для оптимизации
- Анимации запускаются только при монтировании компонентов
- GSAP оптимизирован для 60 FPS

## Кастомизация

Параметры анимаций можно изменить в файлах:
- `DecodeText.js` - настройки декодирования
- `BackgroundGradient.css` - скорость движения градиентов
- `useStaggerAnimation.js` - параметры каскада
